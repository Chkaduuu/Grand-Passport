# Grand Passport
• Plugin მუშაობს მხოლოდ 1.21.x ვერსიებზე

• Plugin-ის ბრძანებები:
• /passport create - (პასპორტის შექმნა)
• /passport set name [სახელი] - (ვწერთ როგორც EN შრიფტით ასევე GE)
• /passport set surname [გვარი] - (ვწერთ როგორც EN შრიფტით ასევე GE)
• /passport set age [ასაკი] - (ვწერთ ციფრებით 1,2,3...)
• /passport set gender [სქესი] - (ვწერთ Male ან Female)
• /passport view <player> - (ბრძანება რომლითაც მოთამაშეებს შეუძლიათ აჩვენონ თავიანთი პასპორტები)
• Permission:grand.pass.view
• /passport reload - (ხელმისაწვდომია OP და Permission მქონდე მოთამაშეებისთვის)
• Permission:grand.pass.reload

• PlaceHolders
• name -> %grand_name%
• surname -> %grand_surname%
• age -> %grand_age%
• gender -> %grand_gender%
• PlaceHolder-ებს აქვს პრობლემა Plugin TAB-თან და არ გამოაქვს გამოსახულება
• უახლოეს განახლებაში გამოვასწორებთ ბაგს
